## Platform Overview

### What is HASIVU?

HASIVU (Health & Smart Integrated Venue for Universities) is a comprehensive school meal management platform that revolutionizes how students order, receive, and enjoy their school meals. The platform combines:

- 🍽️ **Digital Menu Management**: Browse nutritious, compliant meals
- 📱 **Smart Ordering**: Order meals through web app or mobile
- 🏷️ **RFID Delivery**: Contactless meal pickup using RFID tags
- 💳 **Digital Wallet**: Secure payment system for parents and students
- 📊 **Nutritional Tracking**: Monitor dietary compliance and health goals
- 👨‍🍳 **Kitchen Integration**: Streamlined preparation and delivery workflow
- 📈 **Analytics Dashboard**: Insights for administrators and nutritionists

### Key Benefits

- ✅ **Time Saving**: Reduce lunch queue waiting times by 80%
- ✅ **Nutritional Compliance**: Ensure meals meet dietary requirements
- ✅ **Allergen Safety**: Automatic allergen detection and warnings
- ✅ **Cost Efficiency**: Reduce food waste and optimize inventory
- ✅ **Health Monitoring**: Track nutritional intake and health goals
- ✅ **Parental Control**: Parents can monitor and manage meal plans

## Getting Started

### System Requirements

#### For Web Application

- **Browser**: Chrome 90+, Firefox 88+, Safari 14+, Edge 90+
- **Internet**: Stable connection (recommended: 5 Mbps+)
- **Screen**: Minimum resolution 1024x768
- **JavaScript**: Must be enabled

#### For Mobile Application

- **Android**: Version 8.0+ (API level 26+)
- **iOS**: Version 12.0+
- **Storage**: Minimum 50MB free space
- **Internet**: 3G/4G/5G or Wi-Fi connection

### Account Setup Process

#### 1. Initial Registration

```
Step 1: Visit https://app.hasivu.edu.in/register
Step 2: Select your role (Student, Teacher, Kitchen Staff, Admin)
Step 3: Enter school-provided registration code
Step 4: Fill in personal details
Step 5: Set up dietary preferences and allergen information
Step 6: Verify email address
Step 7: Complete profile setup
```

#### 2. RFID Tag Assignment

```
Step 1: Contact school administrator
Step 2: Receive your unique RFID tag/card
Step 3: Link tag to your account through profile settings
Step 4: Test tag functionality at RFID reader

```
Step 1: Navigate to Wallet section in app
Step 2: Link parent's payment method (card/UPI)
Step 3: Set up automatic top-up preferences

Step 5: Verify first transaction
```

### RFID Meal Pickup

#### How RFID Pickup Works

```
Step 1: Receive "Ready" Notification
├─ Check notification on phone
├─ Note pickup location
├─ Grab your RFID tag/card
└─ Head to cafeteria

Step 2: Approach RFID Station
├─ Look for "RFID Pickup" signs
├─ Wait for green light on reader
├─ Hold RFID tag near reader
└─ Listen for confirmation beep

Step 3: Collect Your Order
├─ Reader displays your name and order
├─ Kitchen staff verifies your identity
├─ Receive your prepared meal
├─ Check order contents
└─ Enjoy your meal!

Step 4: After Pickup
├─ Order status updated to "DELIVERED"
├─ Receipt sent to your email
├─ Option to rate and review
└─ Nutritional data added to profile
```

#### RFID Troubleshooting

**Tag Not Recognized?**

- Ensure tag is properly linked in your profile
- Hold tag steady for 2-3 seconds
- Try different angle or position
- Contact kitchen staff if issues persist

**No Active Order?**

- Check if order is confirmed and paid
- Verify you're at correct pickup location
- Ensure order time hasn't expired
- Ask staff to manually verify order

#### Handling Common Issues

**Student Forgot RFID Tag:**

1. Verify student identity with school ID
2. Look up student in teacher dashboard
3. Show order details to kitchen staff
4. Allow manual meal collection
5. Remind student about RFID importance

**Dietary Restriction Violation:**

1. Stop meal distribution immediately
2. Check student's dietary profile
3. Confirm allergen/restriction details
4. Request alternative meal from kitchen
5. Report incident to admin for review

1. Document the exact problem
2. Try alternative pickup methods

5. Submit incident report post-lunch

```

Temperature Control:
├─ Hot food: Maintain above 60°C (140°F)

├─ Sanitize work surfaces between tasks
├─ Clean utensils after each use
├─ Wash cutting boards with bleach solution
├─ Empty garbage regularly
└─ Deep clean at end of each shift

Food Handling:
├─ First In, First Out (FIFO) for ingredients
├─ Label all prepared items with time/date
├─ Store raw and cooked foods separately
├─ Check expiry dates before use
